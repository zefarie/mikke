# Thermodynamique — chapitre 3 : entropie et second principe

Le premier principe (conservation de l'énergie) ne suffit pas : il n'explique
pas pourquoi la chaleur passe spontanément du corps chaud vers le corps froid
et jamais l'inverse. Le second principe introduit l'**entropie** S, fonction
d'état qui mesure le désordre du système.

Pour une transformation réversible : dS = δQ/T. Pour une transformation
réelle (irréversible), l'entropie de l'univers augmente toujours.

Exemple : détente de Joule. Machine de Carnot : rendement maximal
η = 1 − Tf/Tc. À retenir pour le contrôle : les cycles moteurs tournent dans
le sens horaire dans le diagramme (P, V).
